
# linux-scripting-begginers

Welcome to **linux-scripting-begginers**!  
This project is designed for anyone starting out with Bash scripting on Linux.  
Here you'll find example scripts, automation tips, and step-by-step guides for beginners.

## Getting Started

1. Clone this repository.
2. Run `setup.sh` to set permissions and get started.

## Contents

- `examples/basic_commands.sh` &ndash; Basic Linux commands.
- `examples/automation.sh` &ndash; Automate simple tasks.
- More guides/scripts coming soon.
